🍔 Food Delivery Web App

A responsive food ordering web application built using **React + Vite**.  
Users can browse food categories, add items to cart, and view cart count dynamically.

---

Features

- Food category browsing (Pizza, Burger, Biryani, Dessert)
- Add to cart functionality
- Live cart counter update
- Clean and responsive UI design
- Fast performance with Vite

---

Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- HTML & CSS (Inline styling)


---

Future Improvements

- User authentication
- Payment gateway integration
- Backend with database (orders system)
- Order tracking system
